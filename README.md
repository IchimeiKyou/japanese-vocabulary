# 🎌 日语单词记忆大师

基于 **8,074 个日语单词** 的智能记忆应用

---

## 🚀 快速启动

```bash
cd /Users/kyou/code/AgentConstruction_v2/workspace
node server.js
```

访问：**http://localhost:9999**

---

## 📁 核心文件

| 文件 | 说明 |
|------|------|
| `index.html` | 主页面（29KB） |
| `server.js` | Node.js 服务器 |
| `start.sh` | 启动脚本 |
| `word_data.json` | 8,074 个单词数据 |
| `user_progress.json` | 你的学习进度 |
| `日语单词.xlsx` | 原始数据源 |

---

## ✨ 核心功能

### 🎴 卡片学习
- 点击卡片翻转查看中文释义
- 同时显示日文、读音、汉字、中文
- 3D 翻转动画效果

### 🤖 AI 智能解析
- 查询任意单词的详细解释
- 常用短语和例句展示
- 使用注意事项提醒

### 📊 学习统计
- 已掌握/未掌握分类
- 复习队列管理
- 进度百分比显示

### 💾 自动保存
- 每 30 秒自动备份进度
- 本地文件存储
- 数据安全可靠

---

## ⌨️ 快捷键

| 键 | 功能 |
|----|------|
| **Space** | 翻转卡片 |
| **1** | 标记已掌握 |
| **2** | 标记未掌握 |
| **3** | 开始学习新词 |
| **Esc** | 关闭模态框 |

---

## 🎯 使用流程

1. **启动服务** - 运行 `node server.js`
2. **打开浏览器** - 访问 http://localhost:9999
3. **开始学习** - 点击 "开始学习新词"
4. **记忆单词** - 点击卡片查看含义
5. **分类标记** - 已掌握/未掌握
6. **复习巩固** - 定期复习未掌握单词
